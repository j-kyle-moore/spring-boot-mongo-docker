for harbor.rke2-app.km.test:
Create a robot account for the project in harbor
Download the file that is created by harbor
Run the following command on the kubernetes node:
kubectl create secret generic <insert-secret-name> --from-file=<path to harbor-robot-acct.json>
example:
kubectl create secret generic harbor-regcred --from-file=robot$eaddev.json
NOTE: may have to rename the robot file to get rid of the '$'
